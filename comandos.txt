-git init --> Crea carpeta donde git crea sus bases de datos.
-ls [-a(archivos ocultos)]--------> Lista archivos.
-git status -------------> Ver estado del archivo (rastreado o sin rastrear).
-git add nombreArchivo.ext --> Añade el archivo a area de trabajo (para el nombre del archivo escribimos 
			       algunas letras y luego tabulador.
-git commit ------------> Enviar ficheros al area de git. Para añadir mensaje presionamos "i"."Esc" para salir de 
			  añadir mensaje. Luego escribimos ":wq" para guardar los cambios y salir del editor.

-.gitignore ------------->Todo lo que coloquemos en este fichero git lo va a ignorar (carpeta/, fichero.xxx)
-git log [--oneline(forma abreviada)]-----------> Muestra el historial de commits.
-git checkout número commit --> Saltamos al commit indicado.
